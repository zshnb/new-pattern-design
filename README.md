### 设计模式实践
选取一些经典的设计模式，然后参考其在开源项目中的使用方式，模仿实现，计划如下
- 建造者模式

  模仿对象：Lombok的@Builder注解，grpc生成的Java类
- 责任链模式

  模仿对象：Tomcat的Filter
- 工厂模式与单例模式

  模仿对象：Spring的IOC容器
- 策略模式

  模仿对象：Spring的AOP代理，算法包
- 门面模式

  模仿对象：模板引擎Velocity
- 装饰器模式

  模仿对象：Commons-collection的ListUtils，Java的IO
- 观察者模式

  模仿对象：Spring的ApplicationListener
- 命令模式

  模仿对象：Tomcat
